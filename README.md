# Implied Volatility as a Predictor of Market Uncertainty

## Project Overview

This project investigates whether option-implied volatility reflects market expectations and predicts future realised volatility using over **548,000 AAPL option quotes** collected between **2021 and 2023**.

Unlike traditional option pricing projects that focus on visualising volatility surfaces, this project adopts a quantitative research perspective by combining data validation, market analysis and statistical forecasting.



## Research Question

> Can option-implied volatility explain and predict future market uncertainty?



## Research Workflow

- **Stage 1 — Dataset Overview**
- **Stage 2 — Data Quality Assessment**

- **Stage 4 — Market Behaviour Analysis**
- **Stage 5 — Volatility Forecasting**
- **Stage 6 — Discussion**

## Objectives

- Develop a validated option data pipeline to identify unreliable option quotes.
- Analyse volatility smile, skew and term structure across different maturities.
- Construct constant-maturity ATM implied volatility series.
- Evaluate whether implied volatility predicts future realised volatility.
- Compare implied volatility with historical volatility as a forecasting benchmark.

## Expected Contributions

This project aims to:

- provide a robust framework for validating option market data;
- investigate how implied volatility reflects market expectations;
- evaluate whether implied volatility provides useful information for forecasting future realised volatility.
